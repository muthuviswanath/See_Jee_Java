Variables:
	Containers that is used to store a value (literals)
	Literals are constant value that can be assigned to a variable

	In java variables are categorized into two types

	1. Global Variable
	2. Local Variable

If the varaibles are declared directly inside a class then it called as global variable

Syntax:

[access_specifier][access_modfier]<datatype><variablename>[=<literal>];
